# Backend Engineering Playground

A curated collection of backend engineering exercises, coding interview solutions, and implementation examples across multiple programming languages.

This repository is designed to strengthen problem-solving skills, master common algorithmic patterns, and compare idiomatic implementations in different backend technologies.

Current languages:

* Python
* TypeScript (NestJS)
* C#
* Go

Future additions may include Rust and Java.

---

## Goals

* Master the most common algorithmic patterns used in backend interviews.
* Learn idiomatic implementations in multiple backend languages.
* Build a personal backend engineering knowledge base.
* Practice writing clean, maintainable, and production-quality code.
* Prepare for technical interviews and online coding assessments.
